## qisi-cli

``` bash
npx qisi-cli init my-app
cd my-app
npm run mp     //开发小程序
npm run web    //开发 Web
npm run build  //发布 Web
```

## 目录说明


### Vue

```
├─ dist
│  ├─ mp     //微信开发者工具指向的目录，用于生产环境
│  ├─ web    //web 编译出的文件，用于生产环境
├─ build
├─ src       //项目业务代码
```
